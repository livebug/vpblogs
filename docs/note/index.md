> 保存重要的记忆或事件
## **记录**  
   - **解释**：用文字、符号、图像等方式将事件、经验或思想保存下来。  
   - **例诗**：  
     **但愿人长久，千里共婵娟** ——苏轼《水调歌头》  
     通过诗句记录对亲人的思念，体现记录的情感价值。  
   - **内容方向**：以客观为主，记录生活的点滴、旅途的风景或特定事件。  
   - **可能子主题**：  
     - 日常生活记录  
     - 旅行日志  
     - 特殊时刻的纪念（如节日、生日）  
     - 个人习惯养成或挑战记录  
     - 图片或视频故事
