## 时间轴使用说明

### 语法格式
```markdown
::: timeline
- YYYY-MM-DD 标题
  详细内容描述
  可以有多行内容
- YYYY-MM-DD 另一个标题
  另一个事件的详细描述
:::
```

### 功能特性
1. **自动日期排序**：时间轴会自动按日期排序
2. **响应式设计**：适配各种屏幕尺寸
3. **主题兼容**：支持亮色和暗色主题
4. **丰富内容**：支持文本、列表、代码块、引用等

### 最佳实践
- 使用清晰的日期格式（YYYY-MM-DD）
- 为每个事件提供简洁的标题
- 在详细描述中添加关键信息
- 使用图标或表情符号增强可读性
