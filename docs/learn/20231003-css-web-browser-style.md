# css—用户代理样式表

<p>查阅后才知道，<strong><code>用户代理样式表</code>是浏览器（例如，Chrome，Firefox，Edge 等）提供的“默认样式表”</strong>，<strong>用于以满足“一般显示期望”的方式显示页面</strong>。</p>

一般处理方式怎加第一行代码
```html
<!DOCTYPE html>
```

如果不起作用就需要在样式上使用 `!important`
